# Giáo Trình PTIT (Học viện Công nghệ Bưu chính Viễn thông)

Chào mừng bạn đến với kho lưu trữ tổng hợp tài liệu, bài giảng và giáo trình của PTIT. Kho tài liệu này bao gồm các file PDF bài giảng của các môn học thuộc nhiều khối ngành khác nhau, được phân loại cẩn thận thành các thư mục để giúp bạn dễ dàng tìm kiếm và ôn tập.
