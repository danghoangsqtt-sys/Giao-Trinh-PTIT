# Giáo Trình PTIT (Học viện Công nghệ Bưu chính Viễn thông)

Chào mừng bạn đến với kho lưu trữ tổng hợp tài liệu, bài giảng và giáo trình của PTIT. Kho tài liệu này bao gồm gần 200 file PDF bài giảng của các môn học thuộc nhiều khối ngành khác nhau, được phân loại cẩn thận thành các thư mục để giúp bạn dễ dàng tìm kiếm và ôn tập.

## 📚 Danh mục Tài liệu

Tài liệu được chia thành các nhóm chuyên ngành và lĩnh vực sau:

### 💻 Khối ngành Công nghệ & Kỹ thuật
- **[Khoa học máy tính và Mạng](./Khoa%20học%20máy%20tính%20và%20Mạng/)**: Cấu trúc dữ liệu và giải thuật, Hệ điều hành, Hệ thống nhúng, Mạng máy tính...
- **[Phần mềm và Lập trình](./Phần%20mềm%20và%20Lập%20trình/)**: Nhập môn CNPM, Kiến trúc phần mềm, Lập trình C++/Java/Mạng/HĐT.
- **[Khoa học Dữ liệu và AI](./Khoa%20học%20Dữ%20liệu%20và%20AI/)**: Khai phá dữ liệu, Phân tích dữ liệu, Hệ chuyên gia.
- **[An toàn thông tin](./An%20toàn%20thông%20tin/)**: Cơ sở ATTT, Bảo mật hệ thống, Kỹ thuật giấu tin.
- **[Kỹ thuật Điện tử](./Kỹ%20thuật%20Điện%20tử/)**: Vi xử lý, Điện tử số/tương tự, Hệ thống VLSI, PLC.
- **[Viễn thông và Xử lý tín hiệu](./Viễn%20thông%20và%20Xử%20lý%20tín%20hiệu/)**: Xử lý tiếng nói, Phát thanh truyền hình số.
- **[Đồ họa và Đa phương tiện](./Đồ%20họa%20và%20Đa%20phương%20tiện/)**: Kỹ thuật đồ họa các năm.
